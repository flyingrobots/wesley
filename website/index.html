Wesley

Compile your GraphQL schema into a provably safe PostgreSQL backend with zero-downtime migrations, automated tests, and cryptographic deployment proofs.

Why Wesley?

Schema First

Define your domain once in GraphQL. Wesley generates migrations, types, policies, and tests that stay in lockstep.

Zero Downtime

Automated expand/contract migrations include rehearsal plans and pgTAP coverage to keep production safe.

Evidence Trails

SHA-locked HOLMES reports, WATSON verification, and MORIARTY predictions give reviewers proof that changes are ready.

Getting Started

  1. Clone the repository and install dependencies with pnpm install.
  2. Generate example artifacts via pnpm run bootstrap.
  3. Point Wesley at your schema to produce ready-to-ship Postgres migrations.
